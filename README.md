# 記帳軟體
這是一個使用winForm開發的記帳軟體專案
### 功能
    1. 新增/刪除資料
    2. 依日期、Tag、名稱搜尋想看的資料
    3. 統計年、月、日平均

### 專案UI
![image](https://github.com/user-attachments/assets/d8f1b799-1037-487e-a6e9-c40da74734b0)

### Performance
- 1000筆記錄內的搜尋可在100ms內完成
- 10萬筆約需2.5秒左右
