# 記帳軟體
這是一個使用winForm開發的記帳軟體專案
### 功能
    1. 新增/刪除資料
    2. 依日期、Tag、名稱搜尋想看的資料
    3. 統計年、月、日平均

![image](https://github.com/user-attachments/assets/040ce630-8640-4910-8552-aa5a3af56e1c)

### Performance
- 1000筆記錄內的搜尋可在100ms內完成
- 10萬筆約需2.5秒左右
